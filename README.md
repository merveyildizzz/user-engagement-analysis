# User Engagement & Retention Analysis (DAU/WAU)

## Project Overview
This project analyzes user engagement and retention using Google Sheets by calculating key product metrics such as DAU (Daily Active Users), WAU (Weekly Active Users), and stickiness (DAU/WAU).

## Tools Used
- Google Sheets
- Data Analysis
- KPI Calculation

## Key Metrics
- DAU (Daily Active Users)
- WAU (Weekly Active Users)
- Stickiness (DAU/WAU)
- User Demographics (Age Analysis)

## Analysis
- Calculated weekly and daily active users
- Measured user stickiness over time
- Conducted statistical analysis (mean, median, standard deviation, percentiles)

## Insights
- User engagement shows variability across weeks
- Stickiness indicates moderate daily engagement
- User behavior varies across different segments

## Recommendations
- Improve daily engagement with product features
- Increase stickiness through repeated user interaction
- Segment users for targeted strategies
